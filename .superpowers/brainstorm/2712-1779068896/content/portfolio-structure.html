How text moves everywhere on this page (the Obsidian mechanic, reproduced):
• Headlines → split per character; each letter opacity+translate in, duration grows with letter index (left→right cascade) + random 0–675ms jitter per letter → organic, never typewriter.
• Body / paragraphs → split per line; each line mask-slides up on enter.
• A scroll-linked --progress var keeps headlines subtly reacting to scroll after they land.
• Implemented with GSAP SplitText + CSS vars (matches your v16 pure-math/CSS engine).
